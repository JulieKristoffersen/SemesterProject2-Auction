# Semester Prosject: Auction House

Auction House is a responsive and interactive auction platform designed to allow users to see listings, create listings, place bids, and track their credit balance. It provides a seamless way for users to engage in buying and selling items in a secure and user-friendly environment.

# Description

## The project aims to:

- Allow users to create listings with a title, description, deadline, and media gallery.

- Enable registered users to place bids and view the history of bids on listings.

- Provide users with credits to use for bidding and a way to earn credits by selling items.

- Offer a visually appealing and accessible interface with easy navigation.

- Allow non-registered users to browse and search listings.


## This project was created using:

- HTML5

- CSS3 with Tailwind CSS

- JavaScript (ES6+)

- Noroff Auction API - https://docs.noroff.dev/docs/v2/auction-house/listings

- Netlify (for deployment)

# Set up
## Installing

Follow these steps to get the project up and running locally:

- Clone the repository:

git clone [git@github.com:your-username/auction-website.git](https://github.com/JulieKristoffersen/SemesterProject2-Auction.git)


- Open the project directory:

cd auction-website


- Open index.html in your browser to view the site.

- Running

To preview the site on a local server (optional), you can use a lightweight development server like Live Server:

Install Live Server (if you don't already have it):

For Visual Studio Code users: Install the Live Server extension.

Right-click index.html in your editor and select Open with Live Server.
